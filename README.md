#SafiCircles

SafiCircles is a community-focused digital platform designed to strengthen social collaboration, mutual support, and inclusive participation through ibimina with technology.
The platform emphasizes accessibility, transparency, and modern user experience while addressing real community needs.

##Tech Stack
Frontend

React.js

Vite

React Router (v6)

CSS3 (Custom styling & gradients)

Canvas / CSS Gradients for backgrounds

Tools

Git & GitHub

Node.js & npm

VS Code


Installation & Setup
1. Clone the repository
git clone https://github.com/SafiCircles/safiCircles-web

2️. Navigate into the project
cd safiCircles-web

3. Install dependencies
npm install

4️. Run the development server
npm run dev


The app will be available at:

http://localhost:5173
